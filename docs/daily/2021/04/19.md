## React.js npm

``` sh
# 全局安装 react 脚手架 
npm install -g create-react-app

# 用脚手架创建React项目
npx create-react-app react-demo

# 启动项目
npm start
```

## 查看切换 Node.js 版本

``` sh
# 查看安装的 node，及其版本
nvm list

# 安装 node
nvm install 10.18.1

# 卸载 node
nvm uninstall 10.18.1

# 切换 node 版本
nvm use 10.18.1
```
